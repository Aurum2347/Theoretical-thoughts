# Незеритовый слиток (Netherite Ingot)
| Свойство             | Значение           |
|----------------------|--------------------|
| **Редкость**         | Очень редкий       |
| **Возобновляемый**   | Нет                |
| **Складываемый**     | 64                 |
| **Тип**              | Материалы          |
| **Идентификатор**    | `netherite_ingot`  |

## Описание

**Незеритовый слиток** (*англ. Netherite Ingot*) — это самый прочный и мощный материал в игре, используемый для создания экипировки высшего уровня. Предметы из незерита превосходят алмазные по прочности, урону и эффективности, а также обладают уникальным свойством — они **не горят и не плавятся в лаве**.

**Источники получения:**

*   **Крафт**: Единственный способ изготовления. Для этого требуется **4 незеритовых лома** и **4 золотых слитка**, скомбинированных на верстаке. Также можно получить **9 незеритовых слитков**, разделив **незеритовый блок**.
*   **Нахождение в сундуках**: С небольшим шансом может быть найден в сундуках **развалин бастиона** в Нижнем мире.

## Интересные факты

*   Незеритовый слиток был добавлен в игру **12 февраля 2020 года** в снапшоте **20w06a** для Java Edition и в бета-версии **1.16.0.51** для Bedrock Edition как часть крупного обновления **Nether Update** (версия **1.16**).
*   Это **первый полностью новый материал для инструментов и брони**, добавленный в игру за последние 10 лет — со времён появления золота, алмаза и железа в январе 2010 года.
*   В отличие от других слитков, незерит **не существует в реальном мире** — это вымышленный сплав, созданный специально для Minecraft.
*   Незеритовый слиток нельзя использовать напрямую для крафта оружия или брони. Вместо этого он применяется на **кузнечном столе** для улучшения уже существующих **алмазных предметов**, при этом все чары и уровень износа сохраняются.
*   Незеритовые слитки также можно использовать в качестве топлива для **маяков**, наравне с железом, золотом, изумрудами и алмазами. Однако это крайне **не рекомендуется**, так как незерит — один из самых ценных и труднодоступных ресурсов в игре, и его гораздо эффективнее использовать для улучшения снаряжения.
