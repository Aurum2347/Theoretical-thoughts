# Железная кирка (Iron Pickaxe)

| Свойство             | Значение           |
|----------------------|--------------------|
| **Редкость**         | Обычная            |
| **Возобновляемый**   | Да                 |
| **Складываемый**     | Нет                |
| **Тип**              | Инструменты        |
| **Идентификатор**    | `iron_pickaxe`     |

## Описание

**Железная кирка** (*англ. Iron Pickaxe*) — это первый инструмент, открывающий доступ к «среднему» и «высокому» уровню ресурсов в игре. С её помощью можно добывать **золотую**, **алмазную**, **лазуритовую** и **изумрудную руды**.

**Источники получения:**

*   **Крафт**: Создаётся на верстаке из **3 железных слитков** и **2 палок**.
*   **Нахождение в сундуках**:  
    - С вероятностью **12,1 %** — в **Крепости края**,  
    - **25,6 %** — в **кузнице деревни**,  
    - **4,0 %** — в **кладовой крепости**,  
    - **7,0 %** — в **вагонетке заброшенной шахты** (все — по 1 кирке).

### Характеристики

**Примечание**: В Bedrock Edition все инструменты наносят на 1 сердце больше урона, чем в Java Edition.

| Платформа       | Прочность | Урон (сердца) | Урон в секунду (DPS)  |
|-----------------|-----------|----------------|----------------------|
| Java Edition    | 250       | 4 (2 сердца)   | 4.8                  |
| Bedrock Edition | 251       | 5 (2.5 сердца) | 6.0                  |

**Скорость добычи (в секундах)**

| Блок                     | Время добычи |
|--------------------------|--------------|
| Камень, булыжник, угольная руда | 0.5          |
| Железная руда, медь, необработанная медь и т.д. | 0.75          |
| Обсидиан                | 41.7 (Не добывается)         |

> Железная кирка — **минимальный инструмент**, необходимый для добычи **золота**, **алмазов**, **лазурита** и **изумрудов**.

## Интересные факты

*   **Символ перехода**: В достижениях Java Edition за получение железной кирки даётся **«И кирка без дела ржавеет»** — отсылка к поговорке и намёк на то, что игрок уже перерос примитивные материалы.
