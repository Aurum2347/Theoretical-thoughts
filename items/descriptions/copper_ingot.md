# Медный слиток (Copper Ingot)
| Свойство             | Значение           |
|----------------------|--------------------|
| **Редкость**         | Обычный            |
| **Возобновляемый**   | Да                 |
| **Складываемый**     | 64                 |
| **Тип**              | Материалы          |
| **Идентификатор**    | `copper_ingot`     |

## Описание

**Медный слиток** (*англ. Copper Ingot*) — это важный материал, добавленный в игру сравнительно недавно. Он используется для создания декоративных блоков, функциональных предметов, а также медной брони и инструментов.

**Источники получения:**

*   **Переплавка**: Основной способ. Обжиг **медной руды**, **глубинносланцевой медной руды** или **необработанной меди** в печи, плавильной печи или доменной печи.
*   **Крафт**: Может быть создан из **медного блока** (разбитие даёт 9 слитков) или скрафчен из **9 кусочков меди**.
*   **Дроп с мобов**:  
    *   **Утопленник** имеет шанс 11% выронить 1 медный слиток при смерти (шанс увеличивается до 17% при наличии чар «Добыча III»).  
    *   С **Медного голема** гарантированно выпадает 1–3 медных слитка при убийстве.

## Интересные факты

*   Медный слиток был добавлен в игру **20 ноября 2020 года** в снапшоте **20w45a** для Java Edition и официально выпущен в обновлении **Caves & Cliffs: Part I** (версия **1.17**) в июне 2021 года.
*   Первоначально из медного блока можно было получить только **4 слитка**, но уже в снапшоте **21w17a** количество увеличили до привычных **9**.
*   Медь стала первым материалом в игре, подверженным **естественному окислению**: медные блоки со временем темнеют, проходя четыре стадии от ярко-оранжевого до бирюзово-зелёного цвета.
*   До появления дропа с утопленников (снапшот **21w05a**) медный слиток **не был возобновляемым** ресурсом. Именно этот момент сделал его полностью возобновляемым.
*   Начиная с версии **1.21.9** (снапшот **25w31a**), медные слитки стали использоваться для крафта **медной брони**, **инструментов**, **сундука**, **прутьев**, **цепи**, а также **кусочков меди** — нового компонента, который сам может быть переплавлен обратно в слиток.
