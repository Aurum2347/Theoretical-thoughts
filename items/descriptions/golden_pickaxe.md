# Золотая кирка (Golden Pickaxe)

| Свойство             | Значение           |
|----------------------|--------------------|
| **Редкость**         | Обычная            |
| **Возобновляемый**   | Да                 |
| **Складываемый**     | Нет                |
| **Тип**              | Инструменты        |
| **Идентификатор**    | `golden_pickaxe`   |

## Описание

**Золотая кирка** (*англ. Golden Pickaxe*) — самый быстрый по скорости добычи инструмент среди всех, кроме незеритового, но при этом самый хрупкий. Несмотря на высокую эффективность, она **не может добывать руды среднего и высокого уровня**, включая даже золотую руду. По сути, она копает **только те блоки, что доступны деревянной кирке** — но делает это значительно быстрее.

**Источники получения:**

*   **Крафт**: Создаётся на верстаке из **3 золотых слитков** и **2 палок**.
*   **Нахождение в сундуках**:  
    - Может быть найдена в **сундуках разрушенных порталов** в Нижнем мире (часто уже зачарованная).

### Характеристики

**Примечание**: В Bedrock Edition все инструменты наносят на 1 сердце больше урона, чем в Java Edition.

| Платформа       | Прочность | Урон (сердца) | Урон в секунду (DPS)  |
|-----------------|-----------|----------------|----------------------|
| Java Edition    | 32        | 2 (1 сердце)   | 2.4                  |
| Bedrock Edition | 33        | 3 (1.5 сердца) | 3.6                  |

**Скорость добычи (в секундах)**

| Блок                     | Время добычи |
|--------------------------|--------------|
| Камень, булыжник, угольная руда | **0.2**      |
| Железная руда, медь, необработанная медь | **0.3**      |
| Лазуритовая руда               | **0.3**      |
| Золотая руда, алмазная руда, изумрудная руда | **1.25** (блок разрушается, но **ничего не выпадает**) |
| Обсидиан, древние обломки      | **20.85** (блок разрушается, но **ничего не выпадает**) |

> Золотая кирка **не может добывать** золотую, алмазную, лазуритовую, изумрудную руды, обсидиан и древние обломки — они просто исчезают при разрушении. Однако она **копает быстрее железной** все блоки, доступные деревянной кирке.

## Интересные факты

*   **Самая быстрая «бесполезная» кирка**: Золотая кирка разрушает мягкие блоки (камень, уголь, медь) быстрее, чем железная, алмазная и даже незеритовая — но её прочность (всего 32) делает это преимущество почти бессмысленным в выживании.
*   **Парадокс материала**: Несмотря на то, что золото в реальном мире ценно, в Minecraft оно считается **наименее практичным** материалом для инструментов — его лучше использовать для торговли с пиглинами или создания незерита.
*   **Уникальный «невидимый» уровень**: Золотая кирка формально имеет уровень добычи **ниже каменного**, хотя сделана из более «дорогого» материала. Это единственное исключение в логике прогрессии инструментов.
