# Деревянная кирка (Wooden Pickaxe)

| Свойство             | Значение           |
|----------------------|--------------------|
| **Редкость**         | Обычная            |
| **Возобновляемый**   | Да                 |
| **Складываемый**     | Нет                |
| **Тип**              | Инструменты        |
| **Идентификатор**    | `wooden_pickaxe`   |

## Описание

**Деревянная кирка** (*англ. Wooden Pickaxe*) — это самый первый инструмент, который игрок создаёт в начале игры. Она необходима для добычи камня, угля и других базовых ресурсов, которые невозможно получить голыми руками. Несмотря на низкую прочность и эффективность, без неё невозможно продвинуться дальше первых минут игры.

**Источники получения:**

*   **Крафт**: Создаётся на верстаке из **2 любых досок** и **2 палок**.
*   **Нахождение в сундуках**: С вероятностью **75 %** встречается в **бонусном сундуке** при создании нового мира.

### Характеристики

**Примечание**: В Bedrock Edition все инструменты наносят на 1 сердце больше урона, чем в Java Edition.

| Платформа       | Прочность | Урон (сердца) | Урон в секунду (DPS)  |
|-----------------|-----------|----------------|----------------------|
| Java Edition    | 60        | 2 (1 сердце)   | 2.4                  |
| Bedrock Edition | 60        | 3 (1.5 сердца) | 3.0                  |


**Скорость добычи (в секундах)**

| Блок                     | Время добычи |
|--------------------------|--------------|
| Камень, булыжник, угольная руда | 1.15         |
| Железная руда, золотая руда Нижнего мира | 1.15         |
| Медь, необработанная медь      | 1.15         |
| Золотая руда, алмазная руда, лазуритовая руда | **Нельзя добыть** — блоки просто исчезают |
| Обсидиан, древние обломки      | **Нельзя добыть** |

> Деревянная кирка **эффективнее золотой** по прочности (60 против 32).

## Интересные факты

*   Деревянная кирка была добавлена ещё в **Indev 0.31** (**29 января 2010 года**) и с тех пор остаётся неизменной частью начального геймплея.
*   После добычи **3–18 блоков камня или угля** деревянная кирка обычно теряет смысл своего существования, и многие игроки либо **выбрасывают её**, либо **сжигают в печи** как топливо.
*   Одна деревянная кирка может быть использована как топливо для переплавки **ровно одного предмета**.
*   В отличие от более поздних инструментов, деревянная кирка **не может добывать** золотую, алмазную, лазуритовую и другие «средние» и «высокие» руды — они просто не выпадут при разрушении.
*   Несмотря на свою простоту, деревянная кирка — **ключевой элемент** достижения **«Каменный век»** в Java Edition и **«Пора в забой!»** в Bedrock Edition.
